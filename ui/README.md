# Predictive Cold Chain Monitoring UI

This Streamlit dashboard provides a user interface for monitoring, predicting, and managing cold chain logistics in real time.

## Features
- Overview of system status
- Live sensor data visualization
- Predictive analytics and model forecasts
- Alerts and notifications
- System metrics and agent status

## Getting Started
1. Install Streamlit:
   ```powershell
   pip install streamlit
   ```
2. Run the dashboard:
   ```powershell
   streamlit run ui/app.py
   ```
